    Class CropParameters

    Parameters accepted by the asynchronous inference v2 endpoint.

    All fields are optional except modelId.

    Example

    const params = {
      modelId: "YOUR_MODEL_ID",
      alias: "YOUR_ALIAS",
      webhookIds: ["YOUR_WEBHOOK_ID_1", "YOUR_WEBHOOK_ID_2"],
      pollingOptions: {
        initialDelaySec: 2,
        delaySec: 1.5,
      }
    };
